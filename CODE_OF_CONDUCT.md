# Code of Conduct

## Our Pledge

We pledge to make participation in this project in a respectful way.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while asking clarifying questions.
- Give feedback about ideas and code, not people.
- Be patient with contributors who are learning the project.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or restrict participation for behavior that violates this code of conduct.

To report an issue, contact the maintainers through the repository security or maintainer contact channels.
